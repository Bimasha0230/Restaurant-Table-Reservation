# 🍽️ Restaurant Table Reservation Platform

A Full-Stack Web Application developed as a **1st-Year IT Project at SLIIT**. This platform allows customers to check table availability and book reservations in real-time, while providing administrators with structured control over backend operations.

## 🛠️ Tech Stack & Architecture

This project follows the **MVC (Model-View-Controller) architecture**.

* **Backend:** Java (Servlets), Maven
* **Frontend:** JSP, HTML5, CSS3, JavaScript, Bootstrap
* **Database:** MySQL
* **Server:** Apache Tomcat

---

## 🚀 Key Features

* **User Authentication:** Secure Signup, Login, and Password Reset using SweetAlert.
* **Real-time Table Booking:** Dynamic availability checking to prevent double-booking.
* **Profile Management:** Customers can view and update their profiles and active bookings.
* **Admin Panel:** Manage reservations, update menu items, and review customer messages.
